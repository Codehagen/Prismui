## Usage

```tsx
import { Timeline } from "@/components/prismui/timeline"

export default function Example() {
  return (
    <Timeline
      items={[
        {
          date: "2024-01-07",
          title: "New Feature",
          description: "Added dark mode support",
          href: "/changelog/dark-mode"
        },
        {
          date: "2024-01-05",
          title: "Bug Fix",
          description: "Fixed mobile navigation",
          href: "/changelog/mobile-nav"
        }
      ]}
      initialCount={3}
      showMoreText="Load More"
      showLessText="Show Less"
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `items` | `TimelineItem[]` | Required | Array of timeline items to display |
| `initialCount` | `number` | `5` | Number of items to show initially |
| `className` | `string` | - | Additional CSS classes |
| `showMoreText` | `string` | "Show More" | Text for the show more button |
| `showLessText` | `string` | "Show Less" | Text for the show less button |

### TimelineItem Interface

```tsx
interface TimelineItem {
  date: string;        // ISO date string
  title: string;       // Item title
  description?: string; // Optional description
  href?: string;       // Optional link URL
  icon?: React.ReactNode; // Optional icon
}
```

## Features

- Responsive design with mobile-first approach
- Smooth animations using Framer Motion
- Expandable content with "Show More/Less"
- Automatic date sorting (newest first)
- Customizable text and styling
- Support for icons in timeline dots
- Accessible markup with proper ARIA labels
- Optional links for each timeline item
- Hover states and transitions

## Customization

<Steps>
  <Step>Custom Styling</Step>
  ```tsx
  <Timeline
    className="my-custom-timeline"
    items={items}
  />
  ```

  <Step>Custom Button Text</Step>
  ```tsx
  <Timeline
    showMoreText="Load More Events"
    showLessText="Collapse Events"
    items={items}
  />
  ```

  <Step>Initial Item Count</Step>
  ```tsx
  <Timeline
    initialCount={3} // Show only 3 items initially
    items={items}
  />
  ```
</Steps>

## Notes

- Built with Framer Motion for smooth animations
- Uses semantic HTML for better accessibility
- Supports both mobile and desktop layouts
- Automatically sorts items by date
- Handles both linked and non-linked items
- Supports custom styling via className prop

### Tailwind Config 

You also need to update the tailwind config to include the animation keyframes:

<MdxCodeBlock language="tsx" code={`      keyframes: {
        "accordion-down": {
          from: {
            height: "0",
          },
          to: {
            height: "var(--radix-accordion-content-height)",
          },
        },
        "accordion-up": {
          from: {
            height: "var(--radix-accordion-content-height)",
          },
          to: {
            height: "0",
          },
        },
        "testimonial-scroll": {
          from: { transform: "translateX(0)" },
          to: { transform: "translateX(calc(-100% - var(--gap)))" },
        },
      },
      animation: {
        "accordion-down": "accordion-down 0.2s ease-out",
        "accordion-up": "accordion-up 0.2s ease-out",
        "testimonial-scroll":
          "testimonial-scroll var(--duration) linear infinite",
      },`} />

## Features

- Multiple layout variants (scroll, grid)
- Auto-scrolling with configurable direction
- Pause on hover functionality
- Twitter integration
- Responsive design
- Smooth animations with Framer Motion
- Accessible markup
- Dark mode support


## Notes

- The auto-scrolling feature uses CSS animations for smooth performance
- Tweet integration requires the Twitter API
- Grid layout is responsive and adjusts based on screen size
- All animations can be disabled for reduced motion preferences

## Usage

### Manual Scroll

```tsx
import { TestimonialSection } from "@/components/prismui/testimonial-card"

export function ManualScrollExample() {
  return (
    <TestimonialSection
      testimonials={testimonials}
      variant="scroll"
    />
  )
}
```

### Grid Layout

```tsx
export function GridExample() {
  return (
    <TestimonialSection
      testimonials={testimonials}
      variant="grid"
      columns={3}
    />
  )
}
```

### Tweet Integration

```tsx
import { TweetCard } from "@/components/prismui/tweet-card"

function ScrollingTweetRow({ direction = "right" }) {
  return (
    <div className="group flex overflow-hidden py-4 [--duration:120s] [--gap:1.5rem] [gap:var(--gap)]">
      {Array(2).fill(0).map((_, i) => (
        <div
          key={i}
          className={cn(
            "flex shrink-0 justify-around [gap:var(--gap)]",
            "animate-testimonial-scroll",
            direction === "left" && "[animation-direction:reverse]",
            "group-hover:[animation-play-state:paused]"
          )}
        >
          {tweetIds.map((id) => (
            <TweetCard key={idx} id={id} hideMedia />
          ))}
        </div>
      ))}
    </div>
  )
}
```

## Features

- Multiple layout variants (scroll, grid)
- Auto-scrolling with configurable direction
- Pause on hover functionality
- Twitter integration
- Responsive design
- Smooth animations with Framer Motion
- Accessible markup
- Dark mode support

## Customization

<Steps>
  <Step>Custom Animation Speed</Step>
  ```css
  .animate-testimonial-scroll {
    --duration: 120s;
  }
  ```

  <Step>Custom Grid Layout</Step>
  ```tsx
  <TestimonialSection
    variant="grid"
    columns={4} // Change this value
    testimonials={testimonials}
  />
  ```

  <Step>Custom Tweet Row</Step>
  ```tsx
  <ScrollingTweetRow
    direction="left"
    className="[--gap:2rem]" // Adjust spacing
  />
  ```
</Steps>

## Notes

- The auto-scrolling feature uses CSS animations for smooth performance
- Tweet integration requires the Twitter API
- Grid layout is responsive and adjusts based on screen size
- All animations can be disabled for reduced motion preferences

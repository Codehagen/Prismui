## Props

### Card

| Prop      | Type      | Description                                     | Default |
| --------- | --------- | ----------------------------------------------- | ------- |
| hover     | boolean   | Enables hover animation and shadow effect        | false   |
| gradient  | boolean   | Applies a gradient background                    | false   |
| bordered  | boolean   | Adds a border around the card                    | false   |
| className | string    | Additional CSS classes                           | -       |
| children  | ReactNode | Card content                                     | -       |

### Card Subcomponents

The Card component comes with several subcomponents for structured content:

- `CardHeader` - Container for card header content
- `CardTitle` - Card title with proper styling
- `CardDescription` - Muted text for descriptions
- `CardContent` - Main content area
- `CardFooter` - Footer area with flex layout

## Usage

Here's how to use the Card component:

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from "@/components/prismui/card"

export default function Example() {
  return (
    <Card hover gradient bordered>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        Main content area
      </CardContent>
      <CardFooter>
        <Button>Action</Button>
      </CardFooter>
    </Card>
  )
}
```

## Customization

You can customize the card appearance using these techniques:

<Steps>
  <Step>Custom padding</Step>
  ```tsx
  <Card className="p-8">...</Card>
  ```

  <Step>Custom gradient</Step>
  ```tsx
  <Card className="bg-gradient-to-r from-blue-500 to-purple-500">...</Card>
  ```

  <Step>Custom hover effect</Step>
  ```tsx
  <Card className="hover:rotate-1 transition-transform">...</Card>
  ```
</Steps>

## Notes

- Use the `hover` prop for interactive cards that need user attention
- The `gradient` prop adds a subtle depth to the card
- Combine with other components like `Button` or `Image` for rich content cards
